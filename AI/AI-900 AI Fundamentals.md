### Deep Learning
- A subset of ML
- Inspired by Human Int using Artificial Neural Networks (ANN)
- Learns from large amounts of data

Neural Networks consist of layers
1. Input Layer - translates content in numbers
2. Processing Layer - completes the transformations, each layer of processing builds on the previous
3. Output Layer - generates the result 

Architectures
1. Feedforward Networks
2. Convolutional Neural Networks (CNN)
3. Recurrent Neural Networks (RNN)
4. Transformers

| Feature                | Machine Learning                         | Deep Learning                                 |
| ---------------------- | ---------------------------------------- | --------------------------------------------- |
| Learning Approach      | Uses algorithms to learn hidden patterns | Uses Neural Networks to learn hidden patterns |
| Dataset Size           | Smaller                                  | Larger                                        |
| Use Cases              | For simpler tasks with low compexity     | For complex tasks                             |
| Training Time          | Lower                                    | Higher                                        |
| Computing Requirements | CPU or less                              | High Performance with GPUs                    |
