
## Introduction
Using ChatGPT for some practical implementations

- Research, summarize, and translate information
- Proofreading, improving and enhancing text
- Simple utility tools

## Researching

```
What are the top 5 books everybody should have read?
```
Not specific enough

```
I'm only interested in books with topics related to business or personal development. Only provide the titles, no summaries required.
```
 Dive deeper

```
What will I learn in book #3? Please provide a short summary, the page count and year when it was published.
```
